常见问题

如何使用 DecideX 进行决策分析？

+

1. 访问"决策助手"页面，描述您面临的决策问题

2. 系统自动完成意图识别，将问题重写为结构化形式

3. RAG 检索相关知识，成本/风险 Agent 实时搜索最新数据

4. 三个专家 Agent 并行分析，综合 Agent 汇总并收敛判断

5. 输出带 Citation 来源标注的结构化决策报告

什么是 RAG 检索增强？系统是怎么用的？

+

RAG（Retrieval Augmented Generation）即检索增强生成，DecideX 实现了双路 RAG：

  • 知识库 RAG：成本/风险/价值三个专业知识库，BM25+向量混合召回，Self-RAG 过滤，Cohere 精排后注入 Agent
  • 记忆 RAG：存储您的历史决策（Gemini 提取摘要后向量化），下次相似场景自动召回，实现个性化分析

系统能搜索最新信息吗？

+

可以。成本分析 Agent 和风险评估 Agent 都集成了 Web Search 工具（DuckDuckGo），会在需要时自动搜索：

  • 最新房价、贷款利率、物价水平
  • 行业政策变化、公司近况
  • 市场风险预警、实时薪资水平

搜索结果会被标注来源，您可以在 Citation 列表中查看。

支持哪些类型的决策问题？

+

系统支持 7 类意图标签，覆盖：

  • 🏢 职业选择（换工作、跳槽、创业、考研等）
  • 📈 投资理财（股票、基金、股权投资等）
  • 🏠 消费购买（买房、买车、大额消费等）
  • ✈️ 旅行出行（目的地、行程规划等）
  • 📚 学习教育（选课、考证、留学等）
  • 👥 人际关系（合作、借钱、处理关系等）
  • 🔮 通用决策（其他类型）

决策报告中的 [1][2] 是什么意思？

+

这是 Citation 引用机制，表示该论点的知识来源。报告末尾会附完整参考文献列表，标注每条依据的来源类型：

  • 📚 [知识库] — 来自专业知识库文档
  • 🧠 [历史记忆] — 来自您的历史决策记录
  • 🌐 [网络搜索] — 来自实时 Web 搜索结果

为什么分析有时会自动停止？

+

系统内置了决策停止规则（Stopping Rules），在以下情况自动终止分析：

  • 收敛停止：连续两轮 Top1 推荐不变且领先优势 ≥ 12%
  • 低收益停止：本轮新增观点 < 2 个，分析开始重复
  • 硬停止：最多 3 轮分析后强制输出结论

这确保了系统不会陷入无效循环，高效给出结论。

语音输入功能如何使用？

+

在决策助手页面，点击输入框左侧的麦克风图标即可开启语音输入。需要注意：

  • 需要浏览器授予麦克风权限
  • 建议在安静环境下使用
  • 说完后点击"停止录音"即可

使用技巧

📝

描述要具体

包含具体数字（薪资、金额、年限），系统可调用 Web Search 获取更精准的对比数据，分析更可信。

🔄

多次使用积累记忆

每次决策会自动保存到记忆库，使用次数越多，系统越了解您的偏好风格，个性化越精准。

📎

查阅 Citation 来源

报告末尾的参考文献列表说明每条依据的来源，方便您自行核实关键数据，做出更有把握的决定。

🧠

相信意图重写

系统会将您的模糊问题重写为结构化版本，这会让分析更聚焦——若觉得理解有误可直接告知纠正。
