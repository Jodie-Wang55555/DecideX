关于 DecideX

智能决策助手，为您的每一个选择提供专业、科学的多维度分析

项目背景

DecideX 是一款智能决策辅助产品，面向用户在职业选择、投资判断、消费决策和学习规划等场景中的决策困难问题。 通过 Multi-Agent 架构对信息进行结构化整理与多维度分析，帮助用户在复杂信息环境中更高效地做出理性决策。

本项目为学术研究项目，探索 AI 协作决策、检索增强生成（RAG）与大模型推理在实际产品中的结合， 在自建评估集上端到端测试，相比无画像基线版本，决策意图识别准确率从 74% 提升至 91%。

技术架构

🤖

LangGraph Supervisor

多智能体 Supervisor 架构，综合 Agent 动态调度三个专家 Agent 并行协作，设有收敛检测与最大轮次停止规则

🧠

意图识别 + 问题重写

LLM 对输入进行意图分类（7类）、问题重写和关键要素提取，结构化意图标签驱动后续 Agent 路由

📚

双路 RAG 检索

知识库 RAG（3个专业知识库）+ 记忆 RAG（用户历史偏好）；BM25+向量混合召回，RRF 融合，Self-RAG 过滤，Cohere 精排

🌐

实时 Web Search

成本/风险 Agent 工具调用 DuckDuckGo 获取实时房价、利率、行业政策，补充模型训练数据时效盲区

🧬

Multi-representation RAG

每轮决策后 Gemini 提取摘要+偏好标签，双表示存储（摘要用于索引检索，完整内容存 metadata 供召回展示）

📎

Citation 溯源机制

决策建议关键论点自动标注引用来源 [1][2][3]，附完整参考文献列表，每条决策依据均可追溯验证

完整技术流程

1

输入理解与意图识别

用户输入经 LangChain 多轮历史提取，LLM 识别决策类型，将模糊诉求重写为结构化问题，提取3-5个关键决策要素。

2

混合检索与知识注入

BM25 关键词检索 + 向量语义检索双路召回，RRF 融合排序，Self-RAG 相关性过滤，Cohere 精排，将领域知识注入各 Agent 上下文。

3

Multi-Agent 并行分析

成本 Agent（含 Web Search）、风险 Agent（含 Web Search）、价值 Agent（含记忆 RAG）三方并行，综合 Agent 收敛检测后汇总。

4

Chain-of-Thought 决策输出

综合 Agent 按 CoT 步骤推理（评分→制约因素→偏好匹配→推荐），附 Citation 引用标注，决策写入记忆库完成闭环。

大语言模型 Gemini 2.5 Pro · GPT-4o（可切换）
网络搜索 DuckDuckGo Search API
前端 HTML5 · CSS3 · JavaScript（原生）
