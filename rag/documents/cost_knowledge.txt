# 成本评估专业知识库

## 一、成本分类体系

### 1.1 显性成本（可直接量化）
- 货币成本：购买价格、服务费用、订阅费、手续费
- 时间成本：完成任务所需的实际工时 × 单位时间价值
- 资源成本：人力、设备、场地、材料等直接投入

### 1.2 隐性成本（需要识别与估算）
- 机会成本：选择A方案而放弃最优B方案的潜在收益
- 学习成本：掌握新技能、适应新环境所需的时间和精力
- 心理成本：决策压力、焦虑程度、情感消耗
- 维护成本：长期持有或使用所需的持续投入
- 转换成本：从现有方案切换到新方案的一次性损耗
- 协调成本：多人参与时的沟通、协商、等待成本

### 1.3 风险成本（潜在损失）
- 失败风险成本：方案失败时的预期损失 × 失败概率
- 延误风险成本：超时完成带来的额外支出
- 质量风险成本：达不到预期效果时的补救成本

## 二、按决策场景的成本评估规则

### 2.1 购房 / 租房决策
- 购房总成本 = 首付 + 贷款利息总额 + 税费（约3-5%）+ 装修费 + 持有成本（物业/维修）
- 租房总成本 = 年租金 × 居住年数 + 搬家成本
- 关键指标：月供收入比不超过30%为健康线
- 隐性成本提示：购房有流动性成本（卖房周期长）；租房有不稳定性心理成本
- 盈亏平衡点：购房通常需住满5-7年才优于租房（因税费和交易成本摊薄）

### 2.2 职业选择决策
- 薪资要综合计算：基本薪资 + 奖金 + 股权（折现） + 福利价值
- 隐性收益：技能成长价值、人脉价值、品牌背书价值
- 隐性成本：通勤时间成本（每天1小时通勤 ≈ 年损失约250小时）
- 跳槽成本：适应期生产力损耗约3-6个月
- 稳定性溢价：创业/高风险岗位的薪资需有30%以上溢价补偿风险

### 2.3 购车决策
- 全生命周期成本 = 购车价 + 保险（年均1-2万）+ 维修保养（年均5000-1.5万）+ 燃油/电费 + 停车费 - 残值
- 新能源 vs 燃油：前5年新能源使用成本通常低20-30%，但电池折旧需考虑
- 贷款购车利息成本：以10万贷款、3年期为例，利息约1.5-2万
- 隐性成本：时间成本（堵车）、心理成本（限行焦虑）

### 2.4 旅行决策
- 显性成本：交通 + 住宿 + 餐饮 + 门票 + 购物预算
- 隐性成本：工作中断成本、行前准备时间、旅行疲劳恢复时间
- 多人旅行额外成本：协调成本（时间对齐难度）、口味偏好差异导致的妥协成本
- 朋友人数与协调成本正相关：2人协调成本低，4人以上协调成本显著上升
- 旅行质量-成本权衡：高档住宿通常带来30-40%体验提升，但成本上升50-100%

### 2.5 技术选型决策
- 开发成本：人力工时 × 日均工资
- 学习成本：新技术学习曲线，初期效率损耗20-40%
- 维护成本：技术债务积累，长期维护成本可能是初期开发的2-3倍
- 生态成本：社区活跃度低的技术，第三方库支持差，问题解决成本高
- 迁移风险：技术选型一旦确定，后期迁移成本极高（通常为重写成本的50-80%）

### 2.6 投资理财决策
- 实际收益 = 名义收益 - 通货膨胀（约2-3%）- 税费 - 交易成本
- 流动性成本：锁定期内无法变现的机会成本
- 时间价值：复利效应下，提前投资1年的长期价值显著
- 分散化成本：过度分散导致管理复杂度上升，单项投资精力不足

## 三、成本评估方法论

### 3.1 全生命周期成本分析（LCC）
不只看初始成本，要计算整个使用周期的总拥有成本（TCO）
公式：TCO = 初始投入 + Σ(年运营成本) - 残值

### 3.2 成本收益比（C/B Ratio）
- C/B < 1：成本高于收益，不推荐
- C/B = 1-1.5：基本合理
- C/B > 1.5：成本效益良好，推荐

### 3.3 成本敏感度分析
识别成本构成中占比最高的项目（通常20%的成本项贡献80%的总成本）
重点优化高占比成本项，对低占比项不必过度纠结

### 3.4 沉没成本原则
已发生且无法回收的成本不应影响未来决策
常见误区：因为已经投入了时间/金钱，而继续坚持一个明显错误的方案

## 四、成本评估警示规则

- ⚠️ 当月供/租金超过收入40%时，财务压力极大，强烈预警
- ⚠️ 当隐性成本超过显性成本50%时，需重新评估方案实际价值
- ⚠️ 当协调成本占比超过总成本30%时，多人方案需谨慎
- ⚠️ 忽略机会成本是最常见的决策偏差，必须显式列出
- ⚠️ 一次性高成本 vs 持续低成本：注意长期积累效应（复利）
